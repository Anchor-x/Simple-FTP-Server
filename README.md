# Simple-FTP-Server
计算机网络期末大作业，利用Python Socket编程实现一个简单的FTP服务器

## 解释器要求
`Python >= 3.10`

## 运行方式

### 终端运行
```bash
python .\PutServer.py
```

### 再在新终端运行
```bash
python .\GetServer.py
```

### 运行`main.py`
```bash
python .\main.py
```
### 上传文件到服务器
```bash
put 文件地址
```

### 下载文件到本地
```bash
get 文件名
```
